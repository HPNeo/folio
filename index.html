Folio.js

Slide-based web app with jQuery

Requirements

The site must have the following HTML structure: 

<div class="view">
  <div class="page"></div>
  <div class="page"></div>
</div>
<div class="view">
  <div class="page"></div>
  <div class="page"></div>
  <div class="page"></div>
</div>

Usage

var folio = $('#slide_wrapper').folio();
