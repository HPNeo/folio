Folio.js

Slide-based web app with jQuery

Requirements

The site must have the following HTML structure: 

<div class="view">
  <div class="page"></div>
  <div class="page"></div>
</div>
<div class="view">
  <div class="page"></div>
  <div class="page"></div>
  <div class="page"></div>
</div>

Usage

Initialize:

var folio = $('#slide_wrapper').folio();

Get current view:

var view = folio.currentView;

Slide to page:

view.slideTo('#page_2');

Slide to view:

folio.slideTo('#view_2');
